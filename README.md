# gleam

## Usage

Install [boot](https://github.com/tailrecursion/boot). Then run: `boot gleam-app` in a terminal. Load up [localhost:8000](http://localhost:8000) when compilation finishes.

## Technical stuff

* Hoplon and Castra on Ring
* Boot for tasks
